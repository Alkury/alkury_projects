# Sender Gifts

Проект для отправки подарков в Telegram. Состоит из двух частей: **клиент пользователя** (Pyrogram) для отправки подарков по короткой команде и **бот-админ** (Aiogram) для добавления новых подарков в список.

## Возможности

- **Отправка подарков** — в личном чате с собеседником напишите `.название_подарка` или `.название_подарка текст` — подарок отправится в этот чат (текст опционален).
- **Добавление подарков** — через бота админ добавляет ID подарка и короткое имя; после этого подарок можно отправлять командой `.имя`.

## Структура проекта

```
sender_gifts/
├── main.py       # Pyrogram-клиент: слушает личные сообщения и отправляет подарки
├── bot_adder.py  # Aiogram-бот: админ добавляет подарки (id + название)
├── config.py     # Настройки (API-ключи, id админа)
├── ids.json      # Список подарков: название → id подарка
└── requirements.txt
```

## Установка

1. Клонируйте репозиторий и перейдите в папку проекта:

```bash
cd sender_gifts
```

2. Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv venv
source venv/bin/activate   # Linux/macOS
# или: venv\Scripts\activate  # Windows

pip install -r requirements.txt
```

3. Настройте конфигурацию в `config.py`:

- **api_id** и **api_hash** — получите на [my.telegram.org](https://my.telegram.org) (для пользовательского аккаунта).
- **API_KEY** — токен бота от [@BotFather](https://t.me/BotFather).
- **admin_id** — ваш Telegram user ID (узнать можно у бота [@userinfobot](https://t.me/userinfobot)).

4. (Опционально) Создайте `.env` с переменными окружения вместо правки `config.py`:

```
BOT_TOKEN=ваш_токен_бота
ADMIN_CHAT_ID=ваш_telegram_id
```

## Запуск

Нужно запустить **оба** процесса.

### 1. Бот для добавления подарков (админ)

```bash
python bot_adder.py
```

В Telegram напишите боту `/start`, затем отправляйте строки вида:

```
5800655655995968830 мишка14ф
```

где первое число — ID подарка, второе — короткое имя. После этого подарок можно отправлять командой `.мишка14ф` (и при необходимости текст: `.мишка14ф С днём рождения!`).

### 2. Клиент для отправки подарков (ваш аккаунт)

```bash
python main.py
```

При первом запуске потребуется войти в аккаунт (номер телефона, код из Telegram). Далее в **личном чате** с тем, кому хотите отправить подарок, напишите:

- `.название_подарка` — отправить подарок без текста
- `.название_подарка ваш текст` — отправить подарок с текстом

Примеры: `.сердце`, `.мишка14ф Поздравляю!`

## Формат ids.json

Файл хранит соответствие «название → id подарка»:

```json
{
  "gifts": {
    "сердце": "5170145012310081615",
    "мишка14ф": "5800655655995968830"
  }
}
```

Новые записи добавляются через бота (`bot_adder.py`).

## Зависимости

- **Pyrogram** — пользовательский клиент Telegram (отправка подарков).
- **Aiogram** — бот для админа (добавление подарков).
- **aiofiles** — асинхронная работа с файлами.
- **pydantic-settings** — настройки из переменных окружения.

## Важно

- Подарки отправляются только из **личных чатов** (фильтр `filters.private`).
- Команды вида `.название` обрабатываются только когда сообщение отправляете **вы** (проверка `me.id == message.from_user.id`).
- Для работы клиента нужен именно **пользовательский** аккаунт (api_id/api_hash с my.telegram.org), а не только бот.

## Лицензия

Проект для личного использования.
